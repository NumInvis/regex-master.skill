# regex-master

A comprehensive skill for regular expression authoring, explanation, and debugging across programming languages.

## Description

`regex-master` provides expertise for writing, explaining, and debugging regular expressions across all major programming languages. Use it whenever you need to:

- Write regex patterns for validation, extraction, substitution, or splitting
- Explain what an existing pattern does, step by step
- Debug patterns that don't match correctly, perform poorly, or cause catastrophic backtracking
- Translate regex between languages (Python ↔ JavaScript ↔ Java ↔ C#, etc.)
- Optimize regex performance, readability, or robustness
- Look up battle-tested patterns for emails, URLs, phones, IDs, dates, etc.

## What's Inside

| File | Description |
|------|-------------|
| `SKILL.md` | Core skill documentation with workflows, best practices, and language-specific quick reference |
| `references/regex-cheatsheet.md` | Syntax reminder for metacharacters, quantifiers, anchors, assertions, and flags |
| `references/common-patterns.md` | Battle-tested patterns for common validation/extraction tasks |
| `references/language-differences.md` | Detailed differences between regex flavors (Python, JavaScript, Java, C#, etc.) |

## Quick Start

1. Read `SKILL.md` for the full workflows and best practices.
2. Check `references/regex-cheatsheet.md` when you need a syntax refresher.
3. Use `references/common-patterns.md` to find ready-made patterns.
4. Consult `references/language-differences.md` when converting regex between languages.

## Supported Languages

- Python (`re` / `regex`)
- JavaScript / TypeScript (`RegExp`)
- Java (`java.util.regex`)
- C# (`System.Text.RegularExpressions`)
- And other PCRE-compatible engines
